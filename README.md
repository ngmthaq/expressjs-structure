# EXPRESSJS for a lightweight server

## Setup
1. Set up enviroment
```
cp .env.example .env
```

2. Install node modules
```
npm install
```

## For development
1. Fill APP_PORT enviroment in .env (for example: 5000)
```
APP_PORT=5000
```

2. Open development server
```
npm run dev
```
